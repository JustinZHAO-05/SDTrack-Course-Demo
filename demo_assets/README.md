# SDTrack Demo Assets

This directory contains a small FE108 subset for live demonstration without relying on external drive paths.

Included sequences:

- `bike_low`
- `box_hdr`
- `star_mul222`
- `star_motion`

`run_demo.ps1` automatically prefers this directory when all requested sequences are available here. If a requested sequence is not present, the demo falls back to `data/FE108/test`.
